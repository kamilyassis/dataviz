## Sobre o Projeto

Este projeto tem como objetivo explorar técnicas e ferramentas de visualização de dados, utilizando datasets públicos de saúde para criar visualizações informativas e interativas.

## Dados

### INDE - Infraestrutura Nacional de Dados Espaciais
- [dataset 1: internações 2025](https://metadados.inde.gov.br/geonetwork/srv/por/catalog.search#/metadata/55017136-1efd-4f48-80db-885170b6610c)
- [dataset 2: ambulatorios 2025](https://metadados.inde.gov.br/geonetwork/srv/por/catalog.search#/metadata/8481d11e-b109-4433-8d29-b2b9be1b4904)

## Tecnologias

- Python
- Bibliotecas de visualização (matplotlib, seaborn, plotly)
